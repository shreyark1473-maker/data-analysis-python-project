# Data Analysis: Python (Intermediate)

**Description:** Intermediate data cleaning & analysis using pandas. Includes a sample sales dataset and `sales_analysis.py` which:
- Reads and cleans data
- Handles missing values
- Performs monthly aggregation and category-wise analysis
- Plots monthly revenue

**Files**
- `sales_analysis.py` — main analysis script
- `sales_data.csv` — sample dataset

**How to run**
1. Install pandas, matplotlib:
   `pip install pandas matplotlib`
2. Run:
   `python sales_analysis.py`
